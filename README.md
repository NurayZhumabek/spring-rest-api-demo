# Spring REST API Demo

Мини-REST API сервис на **Spring Boot**, демонстрирующий принципы построения серверного приложения с валидацией, DTO, обработкой ошибок и аналитикой данных.

---

##  Что решает проект

Проект решает задачу управления данными сенсоров, измерений атмосферы и пользователей, с возможностью:

- валидировать поступающие данные;
- обрабатывать ошибки корректно;
- собирать данные измерений и делать простую аналитику;

---

##  Бизнес-логика проекта

- **Сенсоры (Sensor):**  
  Система позволяет регистрировать новые сенсоры (датчики). Сенсоры служат источником данных измерений.

- **Измерения (Measurement):**  
  От зарегистрированных сенсоров можно получать измерения - например, значения (температура, влажность и др.), и признак «был ли дождь».  
  Можно получить полный список измерений, список только значений, а также подсчитать, сколько из них были дождливыми днями.

- **Пользователи (Person):**  
  Отдельный модуль, демонстрация CRUD-операций: создание пользователя, получение всех пользователей или конкретного по ID с валидацией входных данных.
